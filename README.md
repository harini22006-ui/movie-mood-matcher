# Movie Mood Matcher

Movie Mood Matcher is a responsive movie recommendation website developed using React.js. The website suggests movies based on the user's selected mood and provides an interactive user experience with a clean UI design.

## Features

- Mood-based movie recommendations
- Responsive website design
- Interactive user interface
- Dynamic React components
- Fast and smooth navigation

## Technologies Used

- React.js
- JavaScript
- HTML5
- CSS3

## Project Objective

The objective of this project is to help users discover movies according to their mood in a simple and user-friendly way while demonstrating front-end development skills using React.
